# agentskillforge.com website

Single-page marketing site built with **Astro + Tailwind**.

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## GitHub Pages
- GitHub Pages expects the build output in `dist/`.
- If deploying under a subpath, set `PUBLIC_PATH` in your workflow and in `astro.config.mjs`.
